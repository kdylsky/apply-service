## ✅ wanted_pre_onboarding

요구사항
- 유저는 채용공고에 지원을 한다.
- 기업은 채용공고를 등록한다.
- 채용공고의 일부 사항을 변경한다.
- 채용공고를 삭제한다.
- 채용공고에서 원하는 내용을 search 한다.
